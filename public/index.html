Wie funktioniert GroupPool?
1

Streamer wählen

Wähle einen Streamer der gerade live ist. Nur live Streamer können ausgewählt werden.

2

Gruppe sammelt

Teile den Pool-Link mit deiner Gruppe. Jeder zahlt seinen Anteil sicher via Stripe.

3

Alert auf Stream 🚀

Sobald das Ziel erreicht ist, erscheint ein Gruppen-Alert live auf dem Stream. Der Streamer sieht alle Namen!

💰 Transparente Gebühren: 5% Service Fee + Stripe Zahlungsgebühr werden aufgeschlagen — der Streamer erhält den vollen Betrag.
